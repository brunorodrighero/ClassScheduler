# Class Scheduler / Agendador de Aulas

**Class Scheduler** is a powerful class scheduling solution. Initially designed to address the specific needs of the Federal Institute of Rondônia (IFRO), the system aims to optimize the allocation of class schedules for educators, taking into consideration their availability and certain constraints.

This project is still in its infancy, and we enthusiastically welcome any contributions!

## Project Overview

The objective of the Class Scheduler is to streamline the class assignment process, making it both efficient and straightforward. The system considers various parameters, including the availability of teachers and specific scheduling constraints, to ensure optimal class scheduling.

### Getting Started

The following instructions will guide you through the process of getting a copy of this project up and running on your local machine for development and testing purposes.

Ensure the following prerequisites are met before you proceed:

- Install the [.NET 6.0 SDK](https://dotnet.microsoft.com/download)
- Install [Visual Studio 2022](https://visualstudio.microsoft.com/downloads/)

1. Clone the repository ```bash git clone https://github.com/brunorodrighero/ClassScheduler
2. Open the ClassScheduler.sln solution in Visual Studio.
3. Press F5 or click on Start Debugging to start the application.

# Agendador de Aulas

O Agendador de Aulas é uma solução poderosa para o agendamento de aulas. Inicialmente projetado para atender às necessidades específicas do Instituto Federal de Rondônia (IFRO), o sistema visa otimizar a alocação de horários de aula para os educadores, levando em consideração sua disponibilidade e certas restrições.

Este projeto ainda está em sua infância, e recebemos com entusiasmo qualquer contribuição!

## Visão Geral do Projeto

O objetivo do Agendador de Aulas é simplificar o processo de atribuição de aulas, tornando-o eficiente e direto. O sistema considera vários parâmetros, incluindo a disponibilidade dos professores e restrições específicas de agendamento, para garantir um agendamento de aulas ótimo.
